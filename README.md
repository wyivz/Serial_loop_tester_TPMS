# Serial_loop_tester_TPMS

**串口通信稳定性测试工具**

A serial communication stability testing tool for RS485 sensors.

---

## 功能特性

- **单次发送**：直接输入 HEX 或文本命令
- **循环发送（时间模式）**：`loop on [间隔秒]` - 按固定间隔发送
- **循环发送（RX触发模式）**：`loop rx` - 收到返回值后立即发送下一次
- **自动统计**：成功率、响应时间(RTT)、信号强度
- **图表生成**：自动生成响应时间和信号强度的折线图
- **CSV 记录**：自动保存每次通信的详细数据

### 环境准备


```bash

```

## 命令说明

| 命令 | 说明 |
|---|---|
| `HEX字符串` 或 `文本` | 单次发送命令 |
| `loop on 1.0` | 按时间间隔循环发送（1.0秒间隔） |
| `loop rx` | RX触发模式：收到返回值后发送下一次 |
| `loop off` | 停止循环发送 |
| `quit` / `Ctrl+C` | 退出程序 |

### 循环配置

启动循环发送时，程序会依次询问：
1. 要循环发送的命令（HEX 或文本）
2. 目标循环次数（0 = 无限循环）

---

## CSV 记录格式

循环发送期间，程序会自动将每次通信数据保存到 CSV 文件：

| 列名 | 说明 |
|---|---|
| 序号 | 发送序号 |
| 时间 | 收到数据的相对时间（HH:MM:SS.ms） |
| ASCII内容 | 解析后的ASCII字符串 |
| 信号强度 | 从响应中解析的信号值 |
| 响应时间(ms) | 发送命令到收到响应的间隔（毫秒） |

**示例文件**：`Record_20250618_150140.csv`

```csv
序号,时间,ASCII内容,信号强度,响应时间(ms)
1,15:01:23.456,101,101,1P00EB6001,02,4B,42,00,00,-55,OK,-55,3423.4
2,15:01:26.879,101,101,1P00EB6001,01,4B,42,00,00,-55,OK,-55,3242.2
3,15:01:30.245,101,101,1P00EB6001,01,4B,42,00,00,-55,OK,-55,3356.8
...
```

CSV 文件在循环开始时自动创建，循环结束时自动关闭。

## 配置说明

### 默认配置

| 参数 | 默认值 | 说明 |
|---|---|---|
| 串口号 | COM6 | 可运行时修改 |
| 波特率 | 9600 | 可运行时修改 |
| 超时时间 | 10秒 | 串口读取超时 |
| 等待完整帧 | 100ms | 收到数据后等待完整帧的时间 |

### 运行时修改

启动时会询问是否修改串口配置，直接回车使用默认值：

```
当前默认串口: COM6
请输入串口号（直接回车使用默认值）:
当前默认波特率: 9600
请输入波特率（直接回车使用默认值）:
```

---

## 响应解析

程序会自动解析响应数据，格式要求：

```
...,<信号强度>,<状态>
```

示例：
```
101,101,1P00EB6001,02,4B,42,00,00,-55,OK
                                    ^^^  ^^
                                信号强度   状态(OK/NOT OK)
```
